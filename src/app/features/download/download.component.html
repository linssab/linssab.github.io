Installation instructions

Read me please!

Distribution packages are available in the download links at the end of this page. Download the v2.0.0 and afterwards install the latest update. Download the version corresponding to your system architecture (32- or 64-bit) and run the setup. Always apply the last patch or update available after installing the software! XISMuS works on its own, however, some features may be not available as they rely on the xraylib package. Check the pre-requisites below for further information.


Some XISMuS features are only available if xraylib is installed. Download any version later than 3.3.0.
Windows 7 users may need to install the latest Visual C++ Redistributable Packages for XISMuS to work properly. It can be downloaded for free here.
